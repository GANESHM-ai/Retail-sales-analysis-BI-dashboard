# Retail-sales-analysis-BI-dashboard

Retail sales analysis dashboard using powerBI
An interactive Power BI dashboard designed to analyze retail sales performance across months, product categories, and customer age groups. The dashboard provides key insights into total sales, quantity sold, pricing trends, and category-wise contributions using dynamic filters.

##  Project Overview
This project focuses on analyzing retail sales data to gain insights into sales performance, customer behavior, and product category trends. An interactive dashboard was developed using **Power BI** to help stakeholders monitor key business metrics and make data-driven decisions.

---

##  Objectives
- Analyze total sales and quantity sold
- Identify monthly sales trends
- Compare performance across product categories
- Understand customer age-based purchasing patterns
- Provide interactive filtering for deeper analysis

---

##  Tools & Technologies Used
- **Power BI** – Dashboard creation & data visualization  
- **Microsoft Excel / CSV** – Data cleaning and preparation  

---

##  Dashboard Features
- **KPI Cards**:  
  - Total Sales  
  - Total Quantity Sold  
  - Price per Unit  

 **Visualizations**:
  - Monthly sales trend (Line chart)
  - Sales by product category (Bar chart)
  - Category-wise contribution (Pie chart)
  - Quantity distribution (Donut chart)
  - Sales by customer age group (Bar chart)

 **Interactive Filters**:
  - Month
  - Quarter
  - Product Category

---

##  Key Insights
- Electronics category generates the highest revenue
- Sales peak during specific months indicating seasonal demand
- Customers aged between 25–40 contribute significantly to total sales
- Balanced contribution observed across product categories
